## Overview

This project demonstrates an Excel-based operational dashboard built using simulated ServiceNow ticket data. The dashboard is designed to track key performance indicators (KPIs), analyze trends, and provide actionable insights for service operations.

---

## 🔍 Key Features

* **KPI Tracking**

  * Total Tickets (Monthly)
  * Average Closure Time
  * SLA Breach %
  * Reopen Rate
  * Month-over-Month (MoM) comparison with business-aware logic

* **Trend Analysis**

  * Monthly ticket volume trend to identify patterns and fluctuations

* **Category-Level Insights**

  * Ticket distribution across categories (e.g., Network, Software, Access)
  * Helps identify high-impact problem areas

* **Insight Layer**

  * Summary of key observations derived from data
  * Focus on operational efficiency and performance gaps

---

## 🛠 Tools & Techniques Used

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Power Query (for data merging and transformation)
* Dynamic formulas (COUNTIFS, AVERAGEIFS, etc.)
* Conditional formatting for KPI indicators

---

## 📈 Dashboard Highlights

* Clean KPI cards with directional indicators (▲ ▼)
* Business-aware metric evaluation (increase/decrease interpretation varies by KPI)
* Structured layout: KPIs → Trends → Category Insights → Summary
* Interactive-ready design (can be extended with slicers)

---

## 💡 Key Insights (Sample)

* Ticket volume decreased MoM, indicating reduced workload
* SLA breaches showed a slight increase, highlighting potential process gaps
* Reopen rate decreased, suggesting improved resolution quality
